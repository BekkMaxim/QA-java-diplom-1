# UI тесты

Проект "Stellar Burgers" представляет собой систему для заказа бургеров. 
Основной компонент - класс Burger, который моделирует бургер, состоящий из булочек и ингредиентов (начинок и соусов).

## Технологии
- Java 11
- JUnit 4
- Mockito
- Maven

## Основные функции
- Создание кастомных бургеров
- Управление составом бургеров
- Расчет стоимости
- Формирование заказа
